# Dr. Andrew David Edwards - Personal Website

Professional academic website for Dr. Andrew David Edwards, historian specializing in money, capitalism, and the American Revolution.

**Live site:** [andrewdavidedwards.com](https://andrewdavidedwards.com)

## Features

- **Homepage** - Hero section with book promotion and bio preview
- **About** - Full biography and academic journey
- **Book** - *Money and the Making of the American Revolution* (Princeton UP)
- **Publications** - Journal articles and academic work
- **Speaking** - Topics and inquiry form for booking engagements
- **Teaching** - Courses and supervision information
- **Contact** - Contact form and academic profiles

## Tech Stack

- **Framework:** Next.js 16 + React 19 + TypeScript
- **Styling:** Tailwind CSS 4
- **Email:** Resend
- **Analytics:** Vercel Analytics
- **SEO:** Schema.org JSON-LD markup

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Environment Variables

Copy `.env.example` to `.env.local` and fill in:

```bash
RESEND_API_KEY=re_xxxxx
```

## Deployment

The site is deployed on Vercel. Push to `main` to trigger automatic deployment.
